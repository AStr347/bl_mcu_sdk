BL MCU SDK 开发指南¶

博流 BL702 系列产品是基于 “SiFive E24 Core” RISC-V 处理器的通用微控制器，具有 BLE 5.0、zigbee 和以太网等网络功能，同时还具备其它丰富的外设。 可被广泛应用于 IoT 和其他低功耗领域中。 支持通过 JTAG 对芯片进行烧写和调试，也支持通过 UART 进行程序烧录。

BL MCU SDK 将为您提供博流 BL70X 系列 MCU 开发的全方位支持。
